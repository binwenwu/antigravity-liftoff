Today, we are bringing **Google Antigravity to Google Cloud customers**, making it possible for enterprises to use Antigravity with their GCP projects and immediately start using the Gemini 3.5 family of models for inference through Gemini Enterprise Agent Platform. This ensures that every session runs under Google Cloud’s enterprise infrastructure, data privacy, compliance, and is covered by the GCP Terms of Service. 

## The Antigravity Enterprise experience

Antigravity features and capabilities now available in Google Cloud are:

* [Antigravity 2.0](https://antigravity.google/blog/introducing-google-antigravity-2-0): Central standalone application for managing, monitoring, and orchestrating agents  
* [Antigravity CLI](https://antigravity.google/blog/introducing-google-antigravity-cli): Lightweight command-line interface for developers to work with agents  
* Authentication with GCP  
* Direct use of Gemini models from Agent Platform 

Google Antigravity brings agentic coding and agent orchestration to all builders, built on the agent harness co-optimized with the Gemini family of models. Integrated with Gemini Enterprise Agent Platform, it gives your teams an additional pathway to quickly build, scale, and orchestrate applications and agents on Google Cloud. 

GCP admins can set up their organization on Antigravity by following more [detailed instructions](https://antigravity.google/docs/enterprise).

## Consumer vs Enterprise

The core product experiences are shared. Same agent harness, same core product surfaces (Antigravity 2.0 and Antigravity CLI), same features.

By logging in with their Google Cloud credentials and accepting the [Google Cloud Terms of Service](https://cloud.google.com/terms/), Google Cloud customers can use their GCP project in Antigravity to make inference through the Gemini Enterprise Agent Platform. This gives all of the benefits of the Google Cloud Terms of Service, including data governance protections. Concretely, enterprise prompts, responses, code, or other such telemetry is not stored outside of your private environments. Billing is done by consumption at Agent Platform pricing as opposed to through a Google AI subscription. 

## Coming Soon

This is just the start. In the near future, expect:

* Google Antigravity in Gemini Enterprise, including centralized administrative controls on various agent capabilities and product features  
* Support for Antigravity extensions and plugins into existing IDE platforms so that enterprise developers can use Antigravity 2.0 and the Antigravity CLI alongside the IDEs that they already use, as opposed to needing to adopt another IDE
